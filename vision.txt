ESP32-CAM Module  Mini TXT Reader Project Document – D.E.V_Darshan
1️⃣ Project Overview
Project Name: D.E.V_Darshan
Developer: Sakshyam Bastakoti – IOT and Robotics Developer
Board: ESP32-CAM
Display:  0.96 oled display (u8g2 library )
Storage: Built-in SD card (SD_MMC)
Vision:
Create a compact, pocketable offline TXT reader using ESP32-CAM without using the camera module. Users can upload .txt files via Wi-Fi portal, navigate and read them easily using only two tactile buttons.
________________________________________
2️⃣ Features
•	Offline TXT reading with built-in SD card
•	Wi-Fi upload portal to add .txt files without removing SD card
•	Two-button interface:
o	Short UP/DOWN → scroll text
o	Hold UP 2s → select file
o	Hold DOWN 2s → back to menu
o	Hold BOTH 2s → open Wi-Fi portal
•	Home Screen: Lists all .txt files, scrollable
•	File Viewer: Shows 5 readable lines at a time (full OLED space)
•	Word wrapping for long lines (~21 characters per line)
•	Stable power management using 5V battery/USB with capacitor protection

6️⃣ Firmware Architecture
1.	System Initialization
o	OLED init
o	SD card mount
o	Button input setup
2.	Modes
o	HOME_MODE → display file list
o	READ_MODE → display file contents
o	PORTAL_MODE → Wi-Fi upload server
3.	Button Handling
o	Short press → scroll
o	Long press → action
o	Both long → portal
4.	Text Handling
o	Line-by-line reading
o	Word wrap 
o	Scroll index for pages
5.	Wi-Fi Portal
o	AP mode (D.E.V AP /Darshan)
o	Portal to upload .txt files
o	Files saved to SD_MMC
________________________________________
7️⃣ Usage Instructions
1.	Power on device → Home Screen shows file list
2.	Use UP/DOWN buttons to navigate
3.	Hold UP 2s → open selected file
4.	Scroll content using UP/DOWN
5.	Hold DOWN 2s → back to file list
6.	Hold BOTH buttons 2s → open Wi-Fi portal
7.	Connect phone/laptop → navigate to 192.168.4.1 → upload .txt files
